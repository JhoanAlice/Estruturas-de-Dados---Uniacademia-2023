Cada arquivo .txt deve conter os números que você deseja ordenar, 
um número por linha. Por exemplo:


5
3
9
1
6

Os dados de cada arquivo .txt serão lidos pela função read_file(), que transformará cada linha em um número inteiro e retornará uma lista de todos os números. Essa lista é então passada para cada algoritmo de ordenação na função main().

Portanto, você pode ter vários arquivos .txt na pasta "datasets", cada um representando um conjunto de dados diferente. A função main() executará todos os algoritmos de ordenação em todos os conjuntos de dados e medirá o tempo que cada algoritmo leva para ordenar cada conjunto de dados.
